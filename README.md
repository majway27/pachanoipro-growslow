## Execution

### Run, Vite

`npm run dev`

### Build, Deploy

`npm run build`

### Deploy Preview

`npm run serve`

### Troubleshooting

### npm run build

- /dist can be deleted
- is regenerated by `npm run build`

#### npm run dev

- /node_modules/.vite can be deleted
- is regenerated by `npm run dev`

#### Console -> Application -> Storage

- 'Clear Site Data' for chrome webtools can be used
- Clears gun db, Manifest
